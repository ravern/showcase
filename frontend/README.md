# Frontend

Hosts the frontend for each device.

## Requirements

- [yarn](https://yarnpkg.com)

## Development

Install all the dependencies. _Note: `yarn` is to be used instead of `npm`._

```bash
$ yarn
```

Configure the `.env` file.

```bash
$ cp .env.sample .env
```

Then, start the development server.

```bash
$ yarn start
```
